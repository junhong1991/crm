该程序由洪俊所写

本程序运行环境：python3.5.4
说明：一个简易的客户管理系统，数据库为了方便使用的是sqlite

博客系统登录页面为/upload_file/
首页展示路径为/index/
已经实现的功能如下：
1. 首先访问/upload_file/，上传一个excel文件，建议上传xls
2. 上传成功后直接跳转到index页面可以看到客户展示资料


下载该程序代码直接用pycharm打开，指定端口运行即可！
